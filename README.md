evented-servers
===============

Small network services built with libev.

Prerequisites
-------------

- [libev](http://software.schmorp.de/pkg/libev.html)
- [boost](http://www.boost.org/)
- Compiler with C++11 support

Build
-----

    # Clang, LLVM
    make clang

    # GCC (>= 4.7)
    make gcc

    # GCC, static linking
    make gcc static=1
